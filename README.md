# 🏥 Hospital Emergency Room Dashboard (Excel)
An interactive Excel dashboard designed to analyze hospital emergency room data, providing insights into patient flow, wait times, admission status, and departmental distribution.


## 🚀 Overview
This project focuses on monitoring and analyzing emergency room operations using Excel. It helps in understanding patient trends, improving service efficiency, and supporting better healthcare decision-making.


## 📌 Key Features
- 🧑‍⚕️ Total patient count and monthly filtering  
- ⏱️ Average patient wait time analysis  
- 😊 Patient satisfaction score tracking  
- 📊 Admission vs Not Admitted analysis  
- 👥 Gender-wise patient distribution  
- 📉 Delay vs On-time attendance status  
- 📈 Age group-wise patient insights  
- 🏥 Department-wise case distribution
